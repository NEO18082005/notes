# **App Name**: Task Master

## Core Features:

- Task Creation: Allow users to add new tasks with titles and descriptions.
- Task Listing: Display all tasks in a clear, organized list.
- Task Editing: Enable users to modify existing tasks.
- Task Completion: Allow users to mark tasks as complete.
- Task Deletion: Enable users to remove tasks from the list.
- Theme Toggle: Allow users to switch between light and dark themes.

## Style Guidelines:

- Primary color (Light Theme): Sky blue (#87CEEB) to promote a sense of calm and focus.
- Background color (Light Theme): Light gray (#F0F8FF) for a clean and unobtrusive backdrop.
- Accent color (Light Theme): Soft lavender (#E6E6FA) for highlighting important actions and elements.
- Primary color (Dark Theme): Dark gray (#333333) for a modern, low-glare interface.
- Background color (Dark Theme): Black (#000000) for maximum contrast and readability.
- Accent color (Dark Theme): Teal (#008080) for highlighting important actions and elements.
- Body and headline font: 'PT Sans', a modern sans-serif font suitable for headlines and body text.
- Use simple, clean icons to represent task status and actions. Adapt icon colors for light and dark themes.
- Maintain a clean, intuitive layout that prioritizes task visibility and ease of interaction. Ensure layout adapts well to both light and dark themes.
- Use subtle animations for task completion and updates. Ensure animations are visible and appropriate in both light and dark themes.
